# Project Statement: Python Password Generator

## Problem Statement

In the digital age, strong and secure passwords are essential for protecting personal and sensitive information. Many users still rely on weak or easily guessable passwords, increasing vulnerability to cyber-attacks.

## Objective

The objective of this project is to create a simple Python-based password generator that:

- Generates strong, random passwords.
- Allows customization for including digits and symbols.
- Ensures at least one character from each selected category is present for better security.

This tool provides users with a quick and secure way to generate passwords that are difficult to crack.
