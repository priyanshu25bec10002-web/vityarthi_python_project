# 🔐 Python Password Generator

A simple and customizable Python program to generate secure passwords. Users can specify the length of the password and choose whether to include digits and symbols for stronger security.

## Features

- Generate passwords of any length.
- Option to include digits and/or special symbols.
- Ensures at least one digit or symbol is included if selected.
- Randomized password for maximum security.
